Çerez Politikası

Son güncelleme: 01.01.2023

Bu Çerez Politikası, Türk Gezileri web sitesinde çerezlerin nasıl kullanıldığını açıklamaktadır. Web sitemizi ziyaret ettiğinizde, çerezlerin cihazınızda saklanmasına izin vermeniz istenecektir.

1. Çerez Nedir?

Çerezler, web sitesini ziyaret ettiğinizde tarayıcınız aracılığıyla cihazınıza indirilen küçük metin dosyalarıdır. Çerezler, web sitesinin düzgün çalışması için gerekli olan temel işlevleri ve güvenlik özelliklerini sağlamak, kullanıcı deneyimini geliştirmek ve web sitesi trafiğini analiz etmek için kullanılır.

2. Kullandığımız Çerez Türleri

Web sitemizde aşağıdaki çerez türlerini kullanıyoruz:

  • Zorunlu Çerezler: Bu çerezler, web sitemizin düzgün çalışması için gereklidir ve sistemlerimizde kapatılamazlar. Genellikle yalnızca sizin tarafınızdan yapılan ve gizlilik tercihlerinizi ayarlama, oturum açma veya form doldurma gibi hizmet taleplerine yanıt olarak ayarlanırlar.
  • Performans Çerezleri: Bu çerezler, ziyaretçilerin web sitemizi nasıl kullandığı hakkında bilgi toplar. Bu çerezler, hangi sayfaların en popüler olduğunu, kullanıcıların sitede nasıl gezindiğini ve karşılaştıkları hata mesajlarını belirlememize yardımcı olur.
  • İşlevsellik Çerezleri: Bu çerezler, web sitemizi daha kişiselleştirilmiş bir deneyim sunmak için kullanılır. Dil tercihleri veya bölge seçimleri gibi bilgileri hatırlamamızı sağlarlar.
  • Hedefleme/Reklam Çerezleri: Bu çerezler, size ilgi alanlarınıza göre reklamlar göstermek ve reklam kampanyalarının etkinliğini ölçmek için kullanılır.

3. Üçüncü Taraf Çerezleri

Web sitemizde, aşağıdaki üçüncü taraf hizmetlerinden gelen çerezleri de kullanabiliriz:

  • Google Analytics: Web sitesi trafiğini analiz etmek için kullanılır.
  • Google AdWords: Reklam kampanyalarının etkinliğini ölçmek için kullanılır.
  • Facebook Pixel: Facebook reklamlarının etkinliğini ölçmek ve hedef kitle oluşturmak için kullanılır.
  • Hotjar: Kullanıcı davranışını analiz etmek için kullanılır.

4. Çerezleri Nasıl Kontrol Edebilirsiniz?

Çoğu web tarayıcısı, çerezleri kabul etmeyi varsayılan olarak ayarlar. Ancak, tarayıcı ayarlarınızı değiştirerek çerezleri reddedebilir veya çerez gönderildiğinde uyarı verecek şekilde ayarlayabilirsiniz. Çerezleri devre dışı bırakmak, web sitemizin bazı özelliklerinin düzgün çalışmamasına neden olabilir.

Çerezleri kontrol etmek veya silmek için tarayıcınızın yardım dosyasına bakabilir veya aşağıdaki bağlantıları ziyaret edebilirsiniz:

  • Google Chrome
  • Mozilla Firefox
  • Internet Explorer
  • Safari

5. Değişiklikler

Bu Çerez Politikasını zaman zaman güncelleyebiliriz. Önemli değişiklikler olması durumunda, web sitemizde bir bildirim yayınlayacağız veya size doğrudan bildirim göndereceğiz.
